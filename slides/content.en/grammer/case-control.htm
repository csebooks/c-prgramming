Decisions Using Switch

Switch is a control statement for making decisions from multiple choices.

Syntax: switch (integer expression) { case constant1: do this; case constant2: do this; default: do this; }

Explanation: It selects a case that matches the integer expression and executes statements within that case and following ones.
